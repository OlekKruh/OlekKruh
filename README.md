## 📚 Projects & Focus
* **Algorithms & Core:** Building custom libraries and implementing fundamental algorithms.
  * [FTLibft](https://github.com/OlekKruh/FTLibft) — Custom C standard library implementation (because why use the standard one when you can reinvent the wheel?).
  * [FTPrintf](https://github.com/OlekKruh/FTPrintf) — Recreating the standard C `printf` function (mostly because I forgot how to "#include <stdio.h>").
  * [FTGetNextLine](https://github.com/OlekKruh/FTGetNextLine) — Function to read a line from a file descriptor.
  * [FTPushSwap](https://github.com/OlekKruh/FTPushSwap) — Data sorting algorithm using two stacks.
  * [FTCodexion](https://github.com/OlekKruh/FTCodexion) — A modified version of the classic Dining Philosophers problem (multithreading and synchronization).
  * [FTFly-in](https://github.com/OlekKruh/FTFly-in) — Graph traversal algorithms with constraints (simulating drones navigating through city nodes with restricted air corridors. Traffic jams exist in the sky too)).
  * [FTPythonCore](https://github.com/OlekKruh/FTPythonCore) — Core Python concepts, including Object-Oriented Programming (OOP) and working with libraries.

* **Graphics & Games:** Console-based rendering, Pygame, and C++ explorations.
  * [pong-raylib](https://github.com/OlekKruh/pong-raylib) — Simple Pong game built to explore C++ and Raylib (because building a AAA title on day one seemed a bit overly ambitious).
  * [FTA-Maze-ing](https://github.com/OlekKruh/FTA-Maze-ing) — Maze generation using various algorithms with rudimentary graphical visualization.

* **Practical Tools & Databases:** 
  * [OperatorTimeSheet](https://github.com/OlekKruh/OperatorTimeSheet) — Python & SQLAlchemy database app designed for CNC workshop employees' reporting.

* **AI Exploration & Others:** Experimenting with the basics of LLMs.
  * [FTCall_Me_Maybe](https://github.com/OlekKruh/FTCall_Me_Maybe) — Implementing restricted answer generation using LLMs (practicing AI containment protocols just in case Skynet goes live).
